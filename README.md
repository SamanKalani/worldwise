# 🌍 WorldWise - Your Personal Travel Tracker

WorldWise is a dynamic, interactive travel journal web application that allows users to log their journeys, bookmark the cities and countries they have visited, and keep track of their travel memories on a beautiful live world map.

🔗 **Live Demo:** [worldwise-flax-iota.vercel.app](https://worldwise-flax-iota.vercel.app)

---

## 🚀 Features

- **Interactive Map:** Click anywhere on the map to automatically detect the city and country, and save it to your travel list.
- **City & Country Management:** View a comprehensive list of all visited cities (with custom notes and dates) and automatically grouped countries with their respective flags.
- **User Authentication:** Fully functional simulation of user login and signup flows, ensuring personalized data tracking for different accounts.
- **Geolocation API:** Instantly locate your current position on the map with a single click to add your current city.
- **Advanced Routing:** Built with complex nested routes, protected routes (preventing unauthorized access to the app), and dynamic URL parameters for seamless navigation.

---

## 🛠️ Tech Stack & Architecture

This project was built using modern front-end tools and state management patterns:

- **Library:** [React.js](https://react.dev/) (Functional components & Modern Hooks)
- **Routing & Navigation:** [React Router v6](https://reactrouter.com/) (Nested Routes, Dynamic Routes, Protected Routes, Query Strings)
- **State Management:** Advanced state handling using the **Context API** paired with **`useReducer`** for clean, predictable architectural patterns.
- **Styling:** **CSS Modules** for scoped, maintainable, and component-specific styling.
- **Map Integration:** [Leaflet Library](https://leafletjs.com/) (`react-leaflet`) for rendering interactive maps and custom markers.
- **Backend Simulation:** REST API simulation using **MockAPI** to handle CRUD operations for users and cities.

## 💻 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites

Make sure you have **Node.js** and **npm** installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/SamanKalani/worldwise.git](https://github.com/SamanKalani/worldwise.git)
   ```
2. Navigate into the project directory:
   cd worldwise

3.Install dependencies:
npm install

4.Run the development server:
npm run dev

Open http://localhost:5173 with your browser to see the result.
